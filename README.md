# Arabic Invitation Site

Arabic one-page invitation card for WhatsApp sharing and RSVP.

## Configure

Update the values at the top of `app.js` before publishing:

- `title`
- `subtitle`
- `host`
- `hijriDate`
- `gregorianDate`
- `time`
- `venue`
- `note`
- `attire`
- `guestsLabel`
- `whatsappNumber`

Use international digits only for `whatsappNumber`, for example `9665XXXXXXXX`.

## RSVP flow

Guests can:

- enter their name
- choose whether they can attend
- send the response through WhatsApp with a prefilled message
