Product flow

One host journey. One guest journey. No more mixing both into the same page.

1. Create event

Set the title, date, venue, and theme once.

2. Add guests

Store names, optional contact details, and a unique invite code per guest.

3. Share links

Send a private guest URL instead of exposing the full event to everyone.

4. Track responses

Review accepted, declined, and pending guests inside a proper dashboard.
